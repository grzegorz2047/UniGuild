uniguild.command.ally - allows to make an ally with the target guild (leader only)
uniguild.command.create - allows to create a guild
uniguild.command.description - allows to show (or change) guilds description
uniguild.command.disband - allows to disband the guild (by a leader)
uniguild.command.enemy - allows to disband the ally from the target guild (by a leader)
uniguild.command.home - allows to teleport to the guild (guild members only)
uniguild.command.info - allows to show informations about the current or the target guild
uniguild.command.invitationaccept - allows to accept guild invitations to the guild (see uniguild.command.invite permission bellow)
uniguild.command.invite - allows to invite player to the guild (by a leader)
uniguild.command.items - allows to show a UI that show needed items for the guild
uniguild.command.kick - allows to kick player from the guild (by a leader)
uniguild.command.leave - allows to leave from the current guild
uniguild.command.list - allows to shows all guilds on this server
uniguild.command.reload - allows to reloads plugin from the /g reload command
